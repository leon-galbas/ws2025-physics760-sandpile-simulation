# Figures

This directory contains all generated figures for the project report.

## Generated figures

Before figures in the subdirectories can be used, they need to be generated by
running `src/main.py`. For further reference, see the
[project report](../report/2026_SimulatingSandpiles_Beck-Galbas.pdf).

## External figures

The figure demonstrating the [local slope](./local_slope.png) idea was taken
from the following textbook:

> Pruessner, G. (2012). _Self-organised criticality: theory, models and
> characterisation_ (Chapter 4.1). Cambridge University Press.
