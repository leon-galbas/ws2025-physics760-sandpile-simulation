The method for simulating and analyzing sandpiles in this project largely follows the 1991 paper by Christensen et al. \cite{christensen1991dynamical}. A sandpile is defined as a d-dimensional cubic CA of width $N$, meaning that the lattice $R$ has $N^d$ entries.

The algorithms described above were implemented in Python. For efficient handling of multi-dimensional we used the PyTorch \cite{paszke2019pytorch} library. All further data analysis was performed using NumPy\cite{harris2020array} for efficient data manipulation and Matplotlib \cite{hunter2007matplotlib} for visualization.
