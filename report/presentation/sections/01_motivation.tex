\begin{frame}{Self-organized criticality}

    \begin{itemize}
        \item Answer: The Ising model needs a fine-tuned parameter (e.g. temperature) to reach criticality, while earthquakes exhibit this behavior in their own. \\
            $\implies$ Self-Organized Criticality (SOC) \pause
        \item SOC naturally explains the widespread appearance of fractal structures and inverse-frequency noise in nature \pause
        \item Scale invariance: events of all sizes happen, no single "characteristic" scale
            $\implies$ Probability distributions of events follow power laws (Pareto distribution)
    \end{itemize}

\end{frame}

\begin{frame}{Why sandpiles?}

    \begin{itemize}
        \item The Bak-Tang-Wiesenfeld (BTW) sandpile model is a very simple model to study SOC (\cite{bak1987self})
        \item Analogy: Slowly dropping sand onto a table causes regular avalanches of various sizes
    \end{itemize}
